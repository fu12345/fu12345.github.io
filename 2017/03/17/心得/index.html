使用hexo搭建博客
2017-03-17

hexo是一个比较流行的框架,基于node.js

需要安装的工具包括Git Nodejs。

  • 安装hexo

    打开Git的安装目录，在这个目录下找到一个叫Git Bash.vbs的文件，打开如下

    然后输入 npm install hexo-cli -g 执行成功安装完成！

  • 创建Github 域名和空间

    注册

    首先你需要注册一个github账号,有的可以略过,注册的用户名将会到影响你的域名！

    创建仓库

    我们需要创建一个仓库(repository) 来存储我们的网站，点击首页任意位置出现的 New repository按钮创建仓库 如下图

    Respository name 中的username.github.io 的username 一定与前面的用户名一致如下图

  • 创建博客

    打开Git的安装目录，在这个目录下找到一个叫Git Bash.vbs的文件打开输入

    git init username.github.

    运行之后会在你的git安装目录下多出一个username.github.io的文件夹

  • 更改配置

    基本配置

    打开新生成的username.github.io文件夹下的 _config.yml文件进行修改

    title: //你博客的名字

    author: //你的名字

    language: zh-Hans //语言 中文

    theme: //安装的主题名称

    deploy:

    type: git //使用Git 发布
    repo: https://github.com/username/username.github.io.git // 刚创建的Github仓库

    主题配置

    为了让自己的博客好看起来我们需要安装一个主题,可以在这里进行挑选
    然后下载一个自己喜欢的主题 解压 将文件夹名字改成 主题名字
    例如:legend 然后放在 username.github.io文件夹下的themes下

    然后在你的主题文件夹下打开 _config.yml修改主题相应的配置,

    最后在你的username.github.io下的_config.yml文件下修改themes为下载的主题的名字

  • 写文章

    在 username.github.io/source/_posts 文件夹下
    新建一个名为 xxx.md 的文件，用 Markdown 书写你的大作吧。

  • 测试

    打开Git 安装目录下的 Git Bash.vbs 文件，输入

    $cd username.github.io
$ hexo s

    然后出现这个页面

    此时就可以在浏览器上输入https://localhost:4000 访问你的博客了

  • 发布

    打开Git 安装目录下的 Git Bash.vbs 文件，输入

    $ cd username.github.io    
$ hexo g && hexo d

    然后会出现这个页面

    然后你输入github的用户名和密码 然后出现这个页面

    现在你就可以浏览器中输入username.github.io访问你的的博客
